# 📚 Book Finder

A modern and responsive **Book Finder** web application that allows users to search for books using the **Open Library API**. Users can browse book details, view cover images, rate books, and save their favorite books for quick access.

---

# 🚀 Features

* 🔍 Search books by title, author, or keyword
* 📖 View detailed book information
* 🖼️ Display book cover images
* ⭐ Rate your favorite books
* ❤️ Add and remove books from Favorites
* 🌙 Dark/Light theme support
* 📱 Fully responsive design
* ⚡ Fast API-based search with dynamic results

---

# 🌐 API Used

This project uses the **Open Library Search API** to fetch book information.

### Search API

```
https://openlibrary.org/search.json?title={book_name}
```

### Example

```
https://openlibrary.org/search.json?title=harry+potter
```

---

# 📸 Application Workflow

1. Open the application.
2. Enter a book title in the search bar.
3. Click the Search button.
4. Browse the list of matching books.
5. Click a book to view more details.
6. Add books to Favorites.
7. Rate books using the rating feature.
8. Access saved books from the Favorites page.
